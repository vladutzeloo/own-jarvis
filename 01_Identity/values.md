# Values

What Vladimir uses to weigh tradeoffs when goals conflict. These are deliberately fewer than goals — three to five durable principles, not a virtues list.

## Core values

> *Fill in: 3-5 principles. Each should be a real tiebreaker, not a platitude.*
>
> Examples (replace with your own):
> - **Compounding over linear.** Prefer work that builds an asset over work that ends with the deliverable.
> - **Honesty before harmony.** Surface disagreements early; don't smooth them over.
> - **Build the thing.** Bias toward making over discussing.

## How Jarvis should use this

When Vladimir is choosing between options that look similar on paper, the values listed here usually break the tie. If a recommendation conflicts with a stated value, name the conflict explicitly rather than recommending around it.
