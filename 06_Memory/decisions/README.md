# Decisions

Notable choices and their reasoning. Use [[99_Templates/decision]] as the template.

## When to log a decision

- The choice has long-term consequences
- The reasoning is non-obvious
- You'd want to remember why if it came up again
- Future-you might reverse it and need to know what was originally weighed

Don't log: trivial daily choices, choices already documented in a project file (link instead).
