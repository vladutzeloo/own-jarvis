# Glossary

Cross-domain terms. Terms specific to one domain belong in that domain's folder.

## Vault-internal

- **MOC** — Map of Content. An index file pointing at related notes.
- **Frontmatter** — the YAML block at the top of every vault file.
- **Wikilink** — `[[file]]` or `[[file|display]]` link inside the vault.
