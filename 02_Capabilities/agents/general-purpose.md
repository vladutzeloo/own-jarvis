# Agent: General Purpose

The default workhorse — multi-step research, complex searches, broad investigations.

## When to use

- Open-ended research questions
- Searching for keywords/files where the first few tries might miss
- Tasks that span the codebase or web
- Anything that doesn't fit a specialized agent

## When NOT to use

- Tight, specific lookups → use `Explore` (faster, cheaper)
- Implementation planning → use `Plan` (more focused output)

## Briefing

The default agent's strength is breadth. Give it room:

- The actual question, not a prescribed series of steps
- Enough context to make judgment calls
- Cap response length if you don't want a long report
