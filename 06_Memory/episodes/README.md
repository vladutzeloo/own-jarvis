# Episodes

What happened on a given day or in a notable session. Daily notes, weekly reviews, meeting notes, post-mortems.

## Naming

`YYYY-MM-DD-<slug>.md`. Examples:

- `2026-05-08-daily.md` — daily review for that day
- `2026-05-08-weekly-review.md` — Friday weekly review
- `2026-05-08-prep-board-meeting.md` — pre-meeting brief
- `2026-05-08-board-meeting.md` — meeting notes
- `2026-05-08-postmortem-outage.md` — incident review
