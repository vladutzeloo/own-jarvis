# Learnings

Things discovered that should outlive the moment. Use [[99_Templates/learning]] as the template.

## Good learning material

- A bug that was sneaky and how it actually broke
- Advice that turned out to work
- A pattern that came up multiple times before you noticed
- Something that surprised you

## Bad material

- Generic best practices ("write tests") — those are knowledge, not learnings
- Things you knew but forgot — those go in [[06_Memory/facts]] or knowledge
