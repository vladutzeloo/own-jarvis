# Weekly Review

Friday afternoon retrospective + next-week plan. 30 minutes.

## Trigger

- Manual: "Run my weekly review"
- Scheduled: every Friday at 4pm via [[02_Capabilities/skills/schedule]]

## Steps

**1. Pull the week.**
   - This week's daily notes from `06_Memory/episodes/`
   - Project updates from `04_Projects/`
   - Any decisions logged this week in `06_Memory/decisions/`

**2. Reflect.**
   - What did I say I'd do this week? (last week's review)
   - What actually happened? Why the delta?
   - One thing that worked. One thing that didn't.

**3. Update goals.** Open [[01_Identity/goals]] — anything to add, remove, sharpen?

**4. Plan next week.**
   - Three outcomes for the week
   - Big rocks scheduled (calendar blocks for deep work)
   - Anything that needs to be unblocked

**5. Tend the vault.**
   - Process [[07_Inbox/README]]
   - Archive any project that's done (`04_Projects/archive/`)
   - Note any new learnings in `06_Memory/learnings/`

**6. Output.** A note in `06_Memory/episodes/<date>-weekly-review.md` summarizing all of the above.

## Verification step

Compare next week's plan to [[01_Identity/goals]] — do the three outcomes ladder up to a goal? If none of them do, that's a signal worth noticing.
