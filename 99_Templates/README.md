# Templates

Reusable starting points. Copy a template into the right folder, rename, fill in.

## Templates

| Template | Use for | Lives in |
|---|---|---|
| [[99_Templates/daily-note]] | Daily review notes | `06_Memory/episodes/` |
| [[99_Templates/weekly-review]] | Weekly review notes | `06_Memory/episodes/` |
| [[99_Templates/meeting-notes]] | Meeting capture | `06_Memory/episodes/` |
| [[99_Templates/research-brief]] | Deep research output | `05_Knowledge/` or project folder |
| [[99_Templates/project]] | New project `_index.md` | `04_Projects/<project>/` |
| [[99_Templates/decision]] | Decision log entry | `06_Memory/decisions/` |
| [[99_Templates/learning]] | Learning entry | `06_Memory/learnings/` |
| [[99_Templates/knowledge-note]] | Knowledge note | `05_Knowledge/domains/<domain>/` |

## Naming the file you copy

- Episodes / decisions / learnings: `YYYY-MM-DD-<slug>.md`
- Projects: `<project-slug>/_index.md`
- Knowledge: `<topic-slug>.md`
