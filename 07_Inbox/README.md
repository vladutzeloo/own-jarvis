# Inbox

Unprocessed capture. Drop things here when you don't want to file them yet.

## What goes here

- Half-formed ideas
- Quick notes from a meeting before they're processed
- Links to read later
- Things Jarvis surfaces that need Vladimir's review

## What does NOT go here

- Long-term reference material — file it in [[05_Knowledge/README]]
- Active project work — file it in [[04_Projects/README]]
- "I don't know if I'll ever read this" → trash it now

## Cadence

Process via [[03_Workflows/productivity/inbox-zero]] at least weekly. Pair with [[03_Workflows/productivity/weekly-review]].

## Anti-patterns

- Letting the inbox grow past one screen
- Re-reading items without deciding
- Filing into "to read later" — that's just slower trash
