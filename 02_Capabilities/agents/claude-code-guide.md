# Agent: Claude Code Guide

Specialized agent for questions about Claude Code (CLI), Claude Agent SDK, and Claude API.

## When to use

- "Can Claude Code do X?"
- "How do I configure hooks / slash commands / MCP servers in Claude Code?"
- "How do I use the Anthropic SDK / API?"
- Building custom agents with the SDK

## When NOT to use

- General coding help (use the parent conversation or `general-purpose`)
- Questions about Cowork specifically (the user-facing app — this agent is for the developer-facing tools)

## Briefing

State the question directly and any context (which model, which SDK version). Ask for code samples when relevant.
