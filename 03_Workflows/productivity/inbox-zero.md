# Inbox Zero

Process [[07_Inbox/README]] until empty. The vault inbox, not email — though the same logic applies.

## Steps

For each item, decide one of four:

**1. Trash it.** No future value. Delete the file (or move to a `_trash/` folder if you want a recovery window).

**2. Do it now.** If it takes under two minutes, just do it. Then trash or file the residue.

**3. File it.** It's reference material. Move to `05_Knowledge/` (durable) or `06_Memory/` (episodic).

**4. Project it.** It's actionable but not in 2 minutes. Move to `04_Projects/<project>/` and define the next concrete step.

## Anti-patterns

- **Re-reading without deciding.** If you've looked at a file twice and not moved it, force a decision.
- **Filing into "to read later".** That's just a slower trash. If you wouldn't actually read it, trash it now.
- **Letting the inbox grow past one screen.** Run this workflow weekly minimum.
